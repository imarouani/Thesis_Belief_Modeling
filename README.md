## Overview

This project estimates CAN-inspired belief–affect–behavior networks (GGM/EBICglasso) across six contexts (waves 55–57 × low/high institutional trust) from the COSMO COVID-19 survey. Estimated network instances are stored in a Belief Knowledge Graph (BKG) for structured comparison. A Conditional Mean Shift (CMS) analysis connects network structure to distributional geometry.

Full theoretical and methodological details are in the thesis.

Run notebooks in order:

## Setup

**Python** (3.9+): `pip install pandas numpy matplotlib pyreadr streamlit altair pyvis`

**R** (4.3+): `install.packages(c("qgraph", "bootnet", "glasso", "dplyr", "readr", "tidyr", "purrr"))`

**Neo4j** (optional): copy `results/networks/kg_exports/` to your Neo4j import directory and run `import_kg.cypher`.

## Interactive tools

## Reproducibility note

Bootstrap results are stochastic — set `set.seed()` in R before running notebook 02 for exact replication. All figures are generated by notebooks 03b and 05.
